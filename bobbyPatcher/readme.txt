Title: How to patch the AutoBod repositiory.
Author: Daniel Ammon
Date: 16.02.16

This is how to patch the AutoBod repository. The patch files are provided
by TH-Nuernberg. With each patch file e.g. "ImagePoint03_ImagePoint03.001.patch" 
comes a python script file e.g. "patchAutoBod.py". To apply the patch, please
run the python script file in a terminal session:

>>> user@pokinixxx: python {pathToPatchFile}/patchAutoBod.py

If patcher is not able to apply the path, please contact support team at TH-Nuernberg.
